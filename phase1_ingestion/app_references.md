# INDMoney App References

- **App Name:** INDmoney: Stocks, MF, FD & More
- **App ID:** `in.indwealth`
- **Play Store URL:** https://play.google.com/store/apps/details?id=in.indwealth&hl=en_IN

## Notes

- The `google-play-scraper` library uses this ID to fetch public reviews.
